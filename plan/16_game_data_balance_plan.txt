1. ÁTTEKINTÉS
================================================================================

A teljes Ashenfall kódbázis (155 script) készen áll az adatfeldolgozásra, de
a res://data/ mappa szinte üres. Jelenleg csak a gem JSON fájlok léteznek.

Hiányzó adat kategóriák:
 - 45 Skill definíció (SkillData Resource-ok)
 - 100+ Alap item definíció (ItemData Resource-ok)
 - Affix pool értékek (AffixData Resource-ok)
 - 8 Set item definíció (SetItemData Resource-ok)
 - 30+ Legendary item definíció egyedi effektekkel
 - Enemy stat táblák biome/szint szerint (EnemyData Resource-ok)
 - Boss stat táblák (BossData Resource-ok)
 - Crafting receptek (CraftingRecipe Resource-ok)
 - Shop készletek (ShopData Resource-ok)
 - XP görbe és szintezési táblák
 - Loot táblák minden enemy/boss/chest típushoz
 - NPC adatok (párbeszédek, quest givers)
 - Biome theme konfigurációk kitöltése
