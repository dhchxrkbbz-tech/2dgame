================================================================================
  1. CLAUDE GDSCRIPT ASSET PIPELINE
================================================================================

  1.1 Eszközök és technológia
  ────────────────────────────

    ┌──────────────────────┬──────────────────────────────────────────────────┐
    │ Eszköz               │ Szerepe                                          │
    ├──────────────────────┼──────────────────────────────────────────────────┤
    │ Claude Opus 4.6      │ GDScript pixel art kód generálása (fő eszköz)   │
    │ Godot Image API      │ Runtime/editor pixel rajzolás (set_pixel stb.)  │
    │ Godot EditorScript   │ Batch PNG export editor módban                  │
    │ Godot ImageTexture   │ Runtime texture cache (generált képekből)       │
    │ Godot SpriteFrames   │ Animáció resource (GDScript-ből feltöltve)      │
    │ PNG export           │ Végleges sprite fájlok (save_png())             │
    └──────────────────────┴──────────────────────────────────────────────────┘

  1.2 Alapelv: Mit csinál Claude?
  ────────────────────────────────
    Claude Opus 4.6 GDScript kódot ír, amely:
      - Létrehoz egy Image objektumot (pl. 64×96 pixel)
      - Pixel-by-pixel vagy draw_rect() hívásokkal megrajzolja a sprite-ot
      - Minden animáció frame-et külön Image-ként generálja
      - SpriteFrames resource-ba pakolja az animációkat
      - Opcionálisan PNG-be exportálja (EditorScript futtatással)

    Példa egy egyszerű sprite generáló függvényre:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  # assassin_generator.gd                                               │
    │  static func generate_idle_frame(frame_idx: int) -> ImageTexture:     │
    │      var img = Image.create(64, 96, false, Image.FORMAT_RGBA8)         │
    │      img.fill(Color(0, 0, 0, 0))  # átlátszó háttér                   │
    │                                                                         │
    │      # Köpeny alap (sötétlila)                                         │
    │      _fill_rect(img, 16, 20, 32, 60, Color(0.18, 0.08, 0.28))         │
    │      # Fej (bőrszín)                                                   │
    │      _fill_rect(img, 22, 4, 20, 18, Color(0.85, 0.70, 0.55))          │
    │      # Kapucni (fekete)                                                 │
    │      _fill_rect(img, 18, 2, 28, 16, Color(0.08, 0.04, 0.12))          │
    │      # Outline (1px fekete)                                             │
    │      _draw_outline(img, Color(0.0, 0.0, 0.0))                          │
    │      # Frame-specifikus légzés offset                                  │
    │      var breath_offset = [0, -1, -1, 0][frame_idx]                    │
    │      img = _shift_image(img, 0, breath_offset)                        │
    │                                                                         │
    │      return ImageTexture.create_from_image(img)                        │
    └─────────────────────────────────────────────────────────────────────────┘

  1.4 Közös helper függvények (base_pixel_art.gd)
  ─────────────────────────────────────────────────
    Minden generator ezeket örökli / statikusan hívja:

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  # base_pixel_art.gd                                                   │
    │  class_name PixelArtBase                                                │
    │                                                                         │
    │  static func fill_rect(img: Image, x: int, y: int,                    │
    │                         w: int, h: int, col: Color) -> void:           │
    │      for px in range(x, x + w):                                        │
    │          for py in range(y, y + h):                                    │
    │              if px >= 0 and py >= 0 and px < img.get_width()          │
    │                 and py < img.get_height():                             │
    │                  img.set_pixel(px, py, col)                            │
    │                                                                         │
    │  static func draw_outline(img: Image, col: Color,                     │
    │                            thickness: int = 1) -> void:               │
    │      # Alpha edge detection + körvonal rajzolás                        │
    │      ...                                                                │
    │                                                                         │
    │  static func draw_circle(img: Image, cx: int, cy: int,                │
    │                           r: int, col: Color) -> void: ...             │
    │                                                                         │
    │  static func draw_line_px(img: Image, x0: int, y0: int,              │
    │                            x1: int, y1: int, col: Color) -> void: ... │
    │                                                                         │
    │  static func apply_palette(img: Image,                                 │
    │                             palette: Dictionary) -> Image: ...         │
    │                                                                         │
    │  static func shift_image(img: Image, dx: int, dy: int) -> Image: ...  │
    │                                                                         │
    │  static func flip_horizontal(img: Image) -> Image: ...                │
    └─────────────────────────────────────────────────────────────────────────┘

  1.5 SpriteFrames összecsomagolás
  ──────────────────────────────────
    ┌─────────────────────────────────────────────────────────────────────────┐
    │  static func build_sprite_frames(                                      │
    │          anim_data: Dictionary) -> SpriteFrames:                       │
    │      # anim_data = { "idle": [tex0,tex1,tex2,tex3], fps: 5,           │
    │      #               "walk": [tex0,...,tex5], fps: 10, ... }          │
    │      var sf = SpriteFrames.new()                                        │
    │      for anim_name in anim_data:                                       │
    │          sf.add_animation(anim_name)                                   │
    │          sf.set_animation_loop(anim_name, anim_data[anim_name].loop)  │
    │          sf.set_animation_speed(anim_name, anim_data[anim_name].fps)  │
    │          for tex in anim_data[anim_name].frames:                      │
    │              sf.add_frame(anim_name, tex)                              │
    │      return sf                                                          │
    └─────────────────────────────────────────────────────────────────────────┘

  1.6 Batch export workflow (EditorScript)
  ─────────────────────────────────────────
    1. Claude megírja az összes generator .gd fájlt
    2. batch_export_tool.gd EditorScript futtatása (Tools → Execute Script)
    3. Minden generator meghívódik, PNG-ek mentése:
       assets/sprites/characters/assassin/idle_0.png
       assets/sprites/characters/assassin/idle_1.png  stb.
    4. SpriteFrames .tres resource-ok automatikus létrehozása
    5. Godot re-import (automatikus, ProjectSettings-ben beállítva)
    6. Sprite filter: Nearest (pixel art – kötelező!)

    ┌─────────────────────────────────────────────────────────────────────────┐
    │  # batch_export_tool.gd (EditorScript)                                 │
    │  @tool                                                                  │
    │  extends EditorScript                                                   │
    │                                                                         │
    │  func _run() -> void:                                                  │
    │      print("=== ASHENFALL PIXEL ART EXPORT START ===")                │
    │      AssassinGenerator.export_all("res://assets/sprites/characters/") │
    │      TankGenerator.export_all("res://assets/sprites/characters/")     │
    │      MageGenerator.export_all("res://assets/sprites/characters/")     │
    │      WorldTilesetGenerator.export_all("res://assets/tilesets/world/") │
    │      # ... stb. minden category                                        │
    │      print("=== EXPORT COMPLETE ===")                                  │
    └─────────────────────────────────────────────────────────────────────────┘

    │      Color(0.18, 0.08, 0.28): Color(0.08, 0.02, 0.18),  # sötétebb   │

  ──────────────────────────────
    - Pixel art stílus, NEM smooth/anti-aliased

  3.4 Osztály-specifikus megjelenés (Claude prompt leírások)
  ────────────────────────────────────────────────────────────

    ASSASSIN (64×96):
      Claude prompt kulcsszavak:
        - Kapucnis köpeny, sötétlila/fekete
        - Két tőr/dagger a kezekben (ezüst pengék)
